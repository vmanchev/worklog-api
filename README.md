# Worklog API
RESTful API for tracking work and time

## Configuration
Setup the following Apache/Nginx environment variables:

```
-----------------------------------
| Name    | Description           |
-----------------------------------
| DB_NAME | Database name         |
-----------------------------------
| DB_USER | Database username     |
-----------------------------------
| DB_PASS | Database password     |
-----------------------------------
| JWT_SECRET | JWT authentication |
-----------------------------------
```
